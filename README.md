# Uber Ride Analytics (SQL + Power BI)

## Project Overview

This project analyzes 150,000 Uber ride bookings using MySQL and Power BI to uncover operational, financial, and customer behavior insights.

The project follows a complete analytics workflow:

CSV Dataset → MySQL Data Cleaning → SQL Analysis → Power BI Dashboard

## Key Findings

### Operational Insights

* Total Bookings: 150,000
* Completed Rides: 93,000 (62%)
* Unsuccessful Rides: 57,000 (38%)

Driver-side cancellations represented the largest operational challenge.

### Revenue Insights

* Total Revenue: ₹47.26 Million
* Average Booking Value: ₹508

### Vehicle Insights

* Auto generated the highest revenue due to higher booking volume.
* Revenue differences were driven primarily by ride volume rather than fare variation.

### Payment Insights

* UPI was the most preferred payment method.
* Digital payments dominated customer transactions.

### Customer Satisfaction

* Average customer rating remained approximately 4.4/5 across all vehicle categories.
